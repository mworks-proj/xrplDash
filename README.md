
# XRPL Dashboard

HTML Browser based XRPL Dashboard used to access xrpl account to make transactions on the xrp ledger. Powered by GemWallet API.






## Deployment

To deploy this project clone repo and install Live Server
v5.7.9

```bash
  click Go Live
```
